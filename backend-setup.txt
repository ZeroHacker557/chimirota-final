## Step 3: Create Backend README.md

# Chimir ota Jome Masjidi - Backend API

Backend API server for the mosque website.

## Features
- Prayer times management
- Events management
- Admin authentication
- Real-time updates with Socket.io
- Settings management

## Deployment
Deploy to Render.com using the provided render.yaml configuration.

## Environment Variables
- NODE_ENV=production
- FRONTEND_URL=https://your-frontend.vercel.app

## Endpoints
- GET /api/prayer-times
- GET /api/events
- GET /api/settings
- POST /api/admin/login
